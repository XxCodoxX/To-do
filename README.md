# React To-do App

This App Contain simple To-do task and use redux persist to save the data to localstorage

You can run this project

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/XxCodoxX/To-do
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the Project
   ```sh
   npm run dev
   ```
